# OWASP Top 10 (2025) — Vulnerability Study Notes

> Description: Documentation of 5 OWASP Top 10 (2025) vulnerabilities with exploitation methods, real-world cases, and mitigations — VortexTech internship task.

A short reference document covering five vulnerabilities from the current OWASP Top 10 (2025) list, prepared as part of an internship task at **VortexTech**.

## Contents

- **Vulnerabilities covered:**
  1. A01:2025-Broken Access Control
  2. A03:2025-Software Supply Chain Failures
  3. A04:2025-Cryptographic Failures
  4. A08:2025-Software or Data Integrity Failures
  5. A09:2025-Security Logging and Alerting Failures

## What's inside each entry

- Plain-language explanation
- How an attacker could exploit it
- A real-world breach example
- Practical prevention / mitigation steps
